# Design Context

## Users

**Who they are:** Trusted individuals with authorized access to a covert OSINT platform, presented under the guise of spreadsheet enthusiasts.

**Their context:** Technical sophistication varies - some may be OSINT professionals, others may be less technical. Access occurs in various environments - private secure spaces, mobile devices, and potentially public locations. The decoy interface must be convincing to casual observers while functioning seamlessly for authorized users.

**Job to be done:** Access the hidden Shadowbroker OSINT dashboard while maintaining plausible deniability. Users need to trigger the hidden authentication smoothly and access the protected system without raising suspicion.

## Brand Personality

**Voice:** Professional, authoritative, slightly boring but impeccably executed.

**Three-word personality:** "corporate, luxury, convincing"

**Emotional goals:**
- **Casual observers:** Should feel reassured they're seeing a legitimate, well-funded organization
- **Authorized users:** Should feel confident in the system's sophistication while appreciating the humor in the decoy
- **Both:** Should experience a sense of professional polish that suggests serious investment and attention to detail

## Aesthetic Direction

**Visual tone:** Fortune 500 professional with a tech startup polish. The interface should look like it cost millions to develop, with the kind of attention to detail that only comes from serious corporate investment.

**References:**
- **Fortune 500 professional:** McKinsey, Deloitte, IBM corporate pages - authoritative, substantial, trustworthy
- **Tech startup polish:** Stripe, Vercel, Linear modern pages - clean, sophisticated, thoughtful interactions
- **Luxury elements:** High-end financial services, private banking interfaces - premium materials, generous whitespace, refined typography

**Anti-references:**
- **Generic SaaS landing pages** - stock photos, overused gradients, templated layouts
- **Obviously fake corporate sites** - poor spacing, inconsistent typography, cheap-feeling interactions
- **Overly tech/cyber aesthetics** - would undermine the "boring corporate" cover story
- **Amateur or rushed design** - nothing should look like it was quickly thrown together

**Theme:** Light mode primary, with the understanding that the hidden OSINT dashboard may have its own aesthetic. The decoy should feel like a legitimate corporate web presence.

## Design Principles

1. **Impeccable Attention to Detail:** Every pixel, every spacing decision, every interaction should feel intentional and refined. No element should feel like an afterthought.

2. **Corporate Authority Through Restraint:** Use whitespace, typography, and subtle interactions rather than flashy effects to convey sophistication and trustworthiness.

3. **Convincing Professionalism:** The design must pass a "real company" test - if someone spent 30 seconds examining it, they should believe this is a legitimate organization with significant resources.

4. **Tech Startup Polish:** While corporate in tone, the execution should have modern smoothness - thoughtful animations, responsive interactions, and contemporary performance.

5. **Accessibility Without Compromise:** Standard accessibility (WCAG AA) should be baked in, not added on. The design should work beautifully for all users while maintaining its aesthetic goals.

## Technical Context

**Framework:** Next.js 14 with TypeScript, React 18
**Styling:** Tailwind CSS with custom design tokens
**Icons:** Lucide React
**Animations:** Framer Motion available
**Current aesthetic:** Emerald/teal gradients, modern corporate feel, responsive design
